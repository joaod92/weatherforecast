# # README.md for Challenge 8 - Weather Forecast App 

Weather forecasting tool - grabbing data from online API and presenting it to user. This applications allows the user to input a location (city) and search for this, promptly displaying the current weather for that location (temperature C, wind speed, humidity) along with forecast for the next 5 days for that location. The app also presents a few location suggestions for the user.  

The web app can be viewed here: (https://joaod92.github.io/weatherforecast)

Here is a screenshot below of the application:

![Demo Screenshot](/weatherforecast.png)


## Installation
No Installation required for this project. Please run the index.html file on your preffered browser of choice, in any device. 

## Usage
Webiste and the related content can be viewed in any browser of choice. Please right-click on the 'index.html' file, move your cursor above the 'Open with' option and select the preffered browser.
